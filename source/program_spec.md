# Program Specification: Combined System Development Services for Rates Exemption Information System (REIS)

## Introduction

This document merges information from two sources: a Program Specification document and a User Requirements Specification (URS) document. It aims to provide a comprehensive overview of the proposed Rates Exemption Information System (REIS) for the Home Affairs Department (HAD).

## 1. Purpose

This document serves as the detailed program specification for all programs within the Rates Exemption Information System (REIS) implementation for the HAD. It outlines the system's purpose, scope, functional and non-functional requirements, technical specifications, and constraints.

## 2. Scope

This specification covers the program codes and software modules produced during the REIS implementation, including:

*   Capture of Rates Exemptions Application
*   Update and amend Rates Exemption Application
*   Uploading of application file
*   Notification of urgent cases for application and review
*   Checking and notification of outstanding cases for application and review
*   Generation of Case Summary
*   Rates Exemption Application Endorsement
*   Generation of Minute list
*   File Approval of Exemption Application
*   Capture of Approved Case Review Information
*   Generation of Random Checking Case
*   Capture of Random Checking Case Results
*   Generation of Matching List Interface file
*   Capture of Matching List Checking Results
*   Generate Memo for application and review
*   Generate Batch Letters
*   Enquiry of Rates Exemption Application Information
*   Enquiry of Rates Exemption Review Case Information
*   Report Generation
*   Access Control
*   Update of System parameter
*   Update of System code table
*   Generation and enquiry of audit log
