# Acceptance Test Plan and Result

This document outlines the Acceptance Test Plan (ATP) and User Acceptance Test (UAT) results for the Licensing Self-Certification Portal (LSCP) system. It covers the objectives, schedule, testing environment, test cases, and results of the user acceptance testing.

## 1. Introduction

This document serves as the User Acceptance Test Plan (UATP) for the Combined System Development Services for Licensing Self-Certification Portal (LSCP) of the Buildings Department.

### 1.1 Objectives of the UAT

The objectives of the UAT are:

- Checking that the delivered functionality works in BD?s specific domain.
- Checking that all functionality required has been delivered.
- Checking that the delivered functionality works according to specified and approved scope and requirements.
