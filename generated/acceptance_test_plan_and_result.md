# Acceptance Test Plan and Result: Licensing Self-Certification Portal & Common Digital Platform for Site Supervision

This document consolidates information from multiple sources to provide a comprehensive Acceptance Test Plan and Result for the Buildings Department's Licensing Self-Certification Portal (LSCP) and the Pilot Project of Common Digital Platform for Site Supervision (CDPSS).

## 1. Introduction

This document outlines the User Acceptance Test Plan (UATP) for the Combined System Development Services for the Licensing Self-Certification Portal (LSCP) and the Pilot Project of Common Digital Platform for Site Supervision (CDPSS) of the Buildings Department.

### 1.1 Objectives of the UAT

The objectives of the UAT are:

*   Checking that the delivered functionality works in BD?s specific domain.
*   Checking that all functionality required has been delivered.
*   Checking that the delivered functionality works according to specified and approved scope and requirements.
