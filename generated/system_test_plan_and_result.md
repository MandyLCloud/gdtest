# System Test Plan and Result for Licensing Self-Certification Portal

## 1. Purpose

This document provides an overview of the system, listing programs, data files, equipment, clerical procedures, and computer operation procedures. It caters to staff responsible for maintaining the application system.

## 2. Scope

This System Test Plan covers the functional, non-functional, and technical features of the Licensing Self-Certification Portal (LSCP). It includes:

-   Function Test of:
    -   Information Framework Website
    -   Institution Website
    -   Administration Module
-   System Integration Test:
    -   System Backup, Fallback, and Recovery Test
    -   Performance, Stress, and Load Test
