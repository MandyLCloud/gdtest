# System Test Plan and Result

This document consolidates information from various sources to provide a comprehensive overview of the System Test Plan and Results for the Licensing Self-Certification Portal (LSCP) project of the Buildings Department.

## 1. Introduction

This document outlines the System Integration Test (SIT) plan for the Combined System Development Services for the Pilot Project of CDPSS of the Buildings Department. It details the scope, methodology, procedures, control, responsible parties, schedule, and other necessary arrangements for the SIT. This plan and specification serve as the baseline for the SIT to prove that the System meets the functional specifications and data migration specifications as defined in the Project Specifications of the Tender.

## 2. Purpose

The purposes of this System Test Plan are:

*   To define the types of tests to be carried out on the system.
*   To define the requirements for test data to be used.
*   To establish the test data which will enable the system as a whole to be tested according to the Plan and Specification.
*   To show the outcome of the system tests, including re-testing carried out after any necessary corrective has been taken.

## 3. Scope

The tests are focused on the functional, non-functional, and technical features of the new Information Framework system.

1.  Function Test of:
    *   Information Framework Web Site
    *   Institution Web-Site
    *   Administration Module
2.  System Integration Test:
    *   System backup, Fall-back and Recovery Test
    *   Performance, Stress and Load Test
